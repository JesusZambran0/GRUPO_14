# Prompt maestro para Claude Code

Actúa como arquitecto senior de machine learning, científico de datos, ingeniero de MLOps y desarrollador full-stack Python. El proyecto debe enfocarse exclusivamente en YouTube y construir un sistema de IA explicable que prediga el potencial de rendimiento de videos, recomiende si deben impulsarse con pauta y sugiera ajustes creativos a partir de métricas públicas, transcripción, OCR y LLM.

## Reglas centrales

- No volver al sector inmobiliario.
- No usar múltiples redes sociales.
- No prometer ROI causal real.
- Entrenar al menos tres modelos y elegir uno por métricas.
- Usar OCR y transcripción como extracción de señales, no como modelos propios entrenados.
- Usar LLM como capa semántico-sintáctica y de recomendación, no como predictor principal.
- Construir demo en Gradio con API y modo seguro.

## Salida esperada

El sistema debe devolver predicción de rendimiento, probabilidad, recomendación de impulso, ajustes sugeridos, métricas, alcance estimado por dólar y justificación.
